## This is a revision of the original Lightboard PCB
This revision includes numerous changes to the PCB. Including:
* USB-C for power delivery (NOTE: This is not capable of data-transfer)
* Refined PCB design
* Larger mounting holes
* An actual power switch

## WANTED CHANGES
* Add JSON to keep the light mode, so when power is returned it is in the same mode as when it turned off

## HARDWARE NEEDED
* Ten (10) 1812 SMD resistors
* Ten (10) Common Cathode LEDs
* Four (4) 3m screws
* One (1) Raspberry Pi Pico
* One (1) 6mm x 6mm Switch (The kind found in Ardunio kits)
* One (1) PCB
* One (1) SPST Slide Switch (6.7x4.1mm W7.62mm P2.54mm)
* One (1) 6 pin USB-C Receptacle
